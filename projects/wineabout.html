WineAbout is an iOS app designed for wine lovers to track, rate, and discover wines. The app features a clean, intuitive interface built with Swift and SwiftUI, making it easy to log tastings, add notes, and explore new favorites.

My Contributions

  • Designed and implemented the front-end using Swift and SwiftUI
  • Created custom UI components for wine entries, ratings, and tasting notes
  • Developed smooth navigation and responsive layouts for iPhone and iPad
  • Integrated animations and accessibility features for a polished user experience
View on GitHub

App Features

  • Track wines you've tasted with detailed notes
  • Easy rating system for quick impressions
  • Discover new wines based on friends' reviews
  • Sleek, modern interface inspired by wine journals
  • Private and social modes for your entries
