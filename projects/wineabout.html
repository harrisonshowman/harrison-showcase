Wine About App

A modern wine journaling and discovery app

Wine About is an iOS app designed for wine lovers to track, rate, and discover wines. The app features a clean, intuitive interface built with Swift and SwiftUI, making it easy to log tastings, add notes, and explore new favorites.

My Contributions

  • Designed and implemented the entire front-end using Swift and SwiftUI
  • Created custom UI components for wine entries, ratings, and tasting notes
  • Developed smooth navigation and responsive layouts for iPhone and iPad
  • Integrated animations and accessibility features for a polished user experience
